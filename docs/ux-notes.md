# UX Notes

## New Primary Flow

Objective: parent reaches first playable game in <= 3 taps.

1. Tap grade.
2. Tap composite band.
3. Tap `Play Now` in Quick Start.

Season remains supported in route and selector but is secondary in hierarchy.

## Results Screen Prioritization

1. **Quick Start** (dominant visual block)
- First game selected from mapped recommendations.
- Buttons: `Play Now`, `Open PDF`, `Print`.
- One short encouragement line.

2. **Minimal interpretation**
- One short paragraph, plain language.

3. **Additional games grouped by skill**
- Skill area cards with direct game launch buttons.

4. **Optional explanation**
- Collapsed details/notes.

## Game Rebuild Improvements

- Larger tappable cards and simplified visual hierarchy.
- Explicit reset/close controls in each game.
- Clear feedback copy (`Correct`, `Try again`, score/round labels).
- Overlay-based game mode with immediate return to results.
- Keyboard-accessible buttons and focusable controls.

## Mobile-First Decisions

- Card-first layout; no dense tables.
- High-contrast band colors and action buttons.
- Reduced copy volume ahead of actions.
